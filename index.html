Welcome

Welcome to the Microsoft Cloud and Enterprise Community Localization project - check out the list of Active Projects!

What is Community Localization?

If you are a Developer and are interested in contributing to an Open Source product it's relatively easy - you download its Sources and write code for a new feature or to fix a bug. (yes, it's a bit oversimplified but you get the point :))

However, if you are not a Developer or are a bit rusty with coding but are fluent in languages other than English, you can still contribute in a significant way by providing or reviewing translations that will make products more usable worldwide by non-English speakers.

To make this easier, Microsoft is using a translator-friendly graphical front end for most localization related work. Translations, reviews, tracking and more can be done without the need to directly access source files or the source code repository.

Engage!

Want to know more? You can join the discussion in the Microsoft Cloud and Enterprise Community Localization LinkedIn Group. You can also reach us on Twitter @microsoftl10n

We just started experimenting with our Gitter chat room. Feel free to drop by and say Hi!

Otherwise, you're welcome to start translating!

Start Translating or Vote for an existing translation

Getting involved is easy:

  1. Go to https://www.transifex.com/microsoft-oss/public/
  2. Sign-up
  3. Request to join a project (we will typically approve within 24 hours)
  4. Start Translating or Vote for your favorite translation. Your vote is equally important as a suggested translation. In fact, in Transifex, a suggestion becomes the officially used translation only if it's supported by a minimum number of votes (usually 2). The suggestion with the most votes becomes the official one.

You can find more detailed instructions here. And here are the shorter instructions on how to contribute to Crowdsourced projects on the Transifex documentation.
